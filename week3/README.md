### 🔹 Roll C – Tooted + inventuur
- 12 toodet ilma müügita  
→ tegelikult dublikaadid → võiks kustutada  

- Sama 12 toodet:
  - puudub ka laoseis  
→ kustutamine on turvaline

- Halvim kategooria:
